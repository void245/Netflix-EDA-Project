# Netflix Data Analysis (EDA Project)


## Project Overview
This project performs **Exploratory Data Analysis (EDA)** on the Netflix Movies and TV Shows dataset to understand content distribution, trends, and key business insights.

The goal is to simulate real-world tasks performed by a **Data Analyst Intern**, including:
- Data Cleaning
- Data Visualization
- Trend Analysis
- Business Insight Generation

---

## Dataset Source
Dataset taken from Kaggle:
Netflix Movies and TV Shows Dataset

## Key Analysis Performed

## Key Insights

- Movies dominate over TV Shows on Netflix.
- Content production increased rapidly after 2016.
- Drama and Comedy are the most common genres.
- The United States produces the most content.
